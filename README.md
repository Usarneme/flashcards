* ReactJS
* React Router
* ExpressJS Server
* HTML
* CSS

### To Test Locally
1. `git clone https://github.com/Usarneme/flashcards.git`
2. `npm install`
3. `npm run start:dev`
4. Navigate to http://localhost:3000 in your browser
